# DSA Problem Solving

Tracking my daily DSA practice.

## Array

- Concatenation of Array  
  Pattern: Index mapping  
  Time: O(n) | Space: O(n)

- Contains Duplicate  
  Pattern: HashSet for duplicate detection  
  Time: O(n) | Space: O(n)

- Valid Anagram  
  Pattern: Fixed-size frequency array (26 lowercase letters)  
  Time: O(n) | Space: O(1)

- Two Sum  
  Pattern: HashMap for complement lookup  
  Time: O(n) | Space: O(n)
